    How this works. There's no trained neural network here — this runs entirely in your browser. It converts your image to a black-and-white silhouette of the text, renders your typed text in the loaded fonts, and scores each by how much their silhouettes overlap (Dice coefficient) plus how closely their proportions match. Local fonts are registered as @font-face rules from the binary data you drop in — nothing is sent to a server. Treat close percentages as "similar shape family" rather than a certified font ID.

    Manifest format. A JSON array of objects with name (display name), file (filename as uploaded), optional cat (sans · serif · display · script · mono) and optional weight (400 · 700). The font binary must be loaded via the Font files tab; the manifest maps names to those files.
